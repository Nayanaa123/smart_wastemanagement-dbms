# SMART WASTE MANAGEMENT SYSTEM

A web-based **Smart Waste Management System** developed using **PHP** and **MySQL** to monitor, record, analyze, and visualize waste generated by colleges or canteens.

The system provides role-based access for **Super Admins** and **Individual Admins**, enabling efficient waste tracking, waste comparison, data visualization, and management through dashboards, charts, alerts, and reports.

This project was developed to promote sustainable waste management practices using digital technologies.

---

# Project Overview

The Smart Waste Management System helps institutions digitally manage food and plastic waste generated in colleges or canteens.

The system allows administrators to:

- Track food waste
- Track plastic waste
- Store daily waste records
- Compare waste levels
- Generate charts and reports
- Monitor waste trends
- View waste reduction recommendations
- Rank colleges/canteens using leaderboard analysis

This project demonstrates concepts related to:

- Web Development
- Database Management
- Data Visualization
- Authentication Systems
- CRUD Operations
- Waste Monitoring Systems

# How to Run the Project

## Step 1 — Install XAMPP

Install XAMPP on your system.

---

## Step 2 — Start Apache and MySQL

Open XAMPP Control Panel and start:

- Apache
- MySQL

---

## Step 3 — Copy Project Folder

Move project folder into:

```bash
C:\xampp\htdocs\
```

Final path:

```bash
C:\xampp\htdocs\WASTE_MANAGEMENT
```

---

## Step 4 — Open phpMyAdmin

```bash
http://localhost/phpmyadmin
```

---

## Step 5 — Create Database

```bash
waste_management
```

---

## Step 6 — Configure Database Connection

Open `db.php`

```php
$conn = new mysqli("localhost","root","","waste_management");
```

---

## Step 7 — Run the Project

```bash
http://localhost/WASTE_MANAGEMENT/login.php
```
